# Gpon-link-budget-calculator
Projeto interdisciplinar de Engenharia de Software e Propagação de Ondas. É uma calculadora de Link Budget para redes GPON com interface gráfica, capaz de isolar a variável faltante automaticamente. O sistema valida os dados com base na norma ITU-T G.984 e separa a lógica da UI usando o padrão MVC.
